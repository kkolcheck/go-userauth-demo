# go-userauth-demo
Authenticate user attempting to log in. Contains single POST /login call. On port :8888

To run, use the following command: go run .

## Background
This was the back end half of a take-home project for an interview with OneCause. It is the Go service that handles the login request. The Angular front end that calls it lives in [angular-login-demo](https://github.com/kkolcheck/angular-login-demo).
